DML - Comandos de manipulação de dados no SQL WorkBench

- `SELECT`: Recuperação de dados específicos.
- `INSERT`: Inserção de novos registros.
- `UPDATE`: Modificação de registros existentes.
- `DELETE`: Exclusão de registros.

DDL - Comandos para definição e alteração da estrutura do banco de dados.

- `CREATE`: Criação de tabelas, bancos de dados e outros objetos.
- `ALTER`: Modificação da estrutura de objetos existentes.

DCL -  Comandos de Controle para definir configurações do banco de dados

## Comandos de Agregação:

Aqui está uma explicação detalhada dos comandos de agregação em SQL que você selecionou:

- **COUNT**: Usado para contar o número de registros em uma tabela ou o número de valores em uma coluna específica. É útil para obter a quantidade total de itens.
- **MIN/MAX**: Funções que retornam respectivamente o menor (MIN) e o maior (MAX) valor de uma coluna. Útil para encontrar valores extremos em conjuntos de dados.
- **AVG (Average)**: Calcula a média aritmética dos valores em uma coluna numérica.
- **SUM**: Soma todos os valores de uma coluna numérica. Como mencionado, é especialmente útil para calcular totais em áreas específicas.
