-Trabalharemos mais com os comandos de DML e DDL

DML - Comandos de manipulação de dados no SQL WorkBench

- `SELECT`: Recuperação de dados específicos.
- `INSERT`: Inserção de novos registros.
- `UPDATE`: Modificação de registros existentes.
- `DELETE`: Exclusão de registros.

DDL - Comandos para definição e alteração da estrutura do banco de dados.

- `CREATE`: Criação de tabelas, bancos de dados e outros objetos.
- `ALTER`: Modificação da estrutura de objetos existentes.

DCL -  Comandos de Controle para definir configurações do banco de dados
